# 8-Week Learning Pathway

This structured pathway organizes the primer's content into a progressive 8-week reading plan aligned with the typical pace of a semester course. Each week focuses on a theme, includes 2-3 readings (~40-60 minutes total), and provides a discussion prompt to deepen engagement.

!!! tip "How to Use This Pathway"
    - Each week's readings build on previous weeks, but individual pages are self-contained
    - Estimated reading times are provided on each page
    - Discussion prompts can be used for self-reflection, study groups, or classroom discussion
    - This pathway covers all 24 content pages across the primer's 5 sections
